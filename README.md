# Лабораторная работа №1

Экран ожидания в стиле DVD, реализованный на C++20 с использованием SFML 3,
ImGui-SFML и nlohmann/json.

## Возможности

- движение логотипа с отражением от границ окна;
- выбор логотипа DVD, Blu-ray или CD;
- настройка скорости, масштаба и цвета через ImGui;
- сброс положения логотипа;
- пауза по пробелу с настраиваемым текстом;
- загрузка начальных параметров из `config.json`.

## Сборка и запуск

```bash
cmake -B build
cmake --build build
cd build/src
./main
```

При сборке CMake автоматически копирует конфигурацию, шрифт и изображения
рядом с исполняемым файлом.
